Collecting local plan data

  1. 1. Install the browser extension

    You can download the extension here

    The browser extension will allow you to collect data for a given local plan. It will

    • Collect urls of the documents related to the local plan
    • Enter the number of houses outlined in the local plan
    • Record any further notes

  2. 2. Start collecting data

    To start collecting data you can either select a Planning Authority from the list or click on the lucky dip button below, this will select a Planning Authority at random for you to look at.
